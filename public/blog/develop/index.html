What its like to be a developer?

Satyam Mahajan
July 22, 2020
eBook

Being a programmer is a hard task, but it is the most amazing thing in this ERA. When I first started coding it was back in 2013 I picked up a book 'C++ For Dummies' till now I remember the feeling the first time I wrote the "Hello World" application.

In this blog we will go over basic issues a programmer faces.

Nowadays, there are tons of JavaScript frameworks from which we might even know the names of many. But whenever we google a problem a solution comes may it be a framework or code snippet.

Do you know all frameworks?

Answer to this question by most of the programmers would be NO! The reason behind this is, everybody a framework it's being built by someone so that the documentation for our code is really important. Because it allows other programmers to use our code easily. Thus, developing an API for everybody is a favour to humanity.

What and how to learn?

This question is intriguing to everyone whether they should learn Java, Python, C++, PHP, and many more. The answer is, "it doesn't matter" whether you learn Java or C++. The main thing you should know the basics of most languages because some features might be available in one language or one in another.

Now the question is where to learn, the documentation of that language is the first step but if you are still unable to grasp it. There are online courses that you can take, a website udemy.com provides a vast range of courses.
