HTML Validator

Features

  • Program will validate HTML file based of opening and closing tags and self closing tags
  • Check the file based of the grammar rules of HTML

How to use

  • Simple compile the program validator.cpp using c++11 compiler
  • Run the program
  • Enter the file name with .html as extension
  • See the result's

Challenges & Solutions

Recursive approach

The main goal was to validate the HTML file and provide appropriate error with the line number

The solution was very simple, follow recursive to parse the file and use a stack to keep track of HTML tags.

How it works

The program parses the file and looks for the HTML tags, The tags are placed in a stack, and the program uses a recursive approach to generate the error if next closing tag doesn't match the last opening tag.
